# Support

MCP Multiplex is an early open source project maintained by its author.

## Community Support

- Use GitHub Issues for reproducible bugs and focused feature requests.
- Include `mxp --version`, operating system, Python version, agent/client name,
  and the exact command that failed.
- Redact tokens, local private paths, private agent configs, audit databases,
  and credential-bearing logs.

## Security Reports

Do not open public issues for suspected vulnerabilities, credential exposure, or
unsafe mutation behavior. Use the repository Security tab and follow
[`SECURITY.md`](SECURITY.md).

## Commercial Support

Commercial support, migration help, compatibility certification, and
organization-level features may be offered separately as the project matures.
